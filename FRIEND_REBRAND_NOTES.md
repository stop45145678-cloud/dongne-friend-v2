# 친구 찾기 앱 리브랜딩 수정 내역

## 앱 방향

- 데이팅/매칭 앱 느낌을 제거하고 `동네친구` 콘셉트로 변경했습니다.
- 핵심 감정 키워드를 하트/호감/설렘에서 일상/공감/대화/매너로 바꿨습니다.
- 메인 흐름은 `모멘츠 → 5분 대화 → 익명 한마디` 순서로 유지했습니다.

## 주요 수정

- 앱 이름: Heart Signal → 동네친구
- 랜덤 통화 → 5분 대화
- 하트시그널 랜덤편지 → 익명 한마디
- 관심 있어요 → 공감해요
- 랭킹 → 매너 온도 랭킹
- 상점 → 표현 상점
- 친구 목록 설명을 스와이프 기반에서 대화/공감 기반으로 변경
- 핑크/하트톤 UI 색상을 그린/뉴트럴톤으로 변경
- README, index.html, Capacitor 앱 이름, 서버 기본 문구 수정

## 확인 사항

- `node --check server/index.js` 문법 검사는 통과했습니다.
- 현재 작업 환경에는 npm 의존성이 설치되어 있지 않아 `vite build`는 바로 실행되지 않았습니다. 사용 환경에서 `npm install` 후 `npm run build`로 확인하면 됩니다.

## 2026-04 통합 업데이트

- 모멘츠는 초기 안정성을 위해 사진 전용으로 변경했습니다. 최대 5장, 파일당 5MB 이하입니다.
- 프로필 수정 기능을 홈에 추가했습니다. MBTI, 대화 스타일, 성격, 취향, 취미, 한줄 소개를 수정할 수 있습니다.
- 모멘츠 본인 게시물 수정/삭제 기능을 추가했습니다.
- 모멘츠 신고 기능을 추가했습니다. 동일 사용자는 같은 모멘츠를 한 번만 신고할 수 있습니다.
- 신고 1회 접수 시 작성자 매너 온도 -1, 신고 3회 누적 시 자동 숨김 + 매너 온도 -5가 적용됩니다.
- 5분 대화 평가에서 1~2점의 낮은 평가를 받은 경우 온도가 내려가도록 추가했습니다. 1점은 -3, 2점은 -1입니다.

## 2026-04-27 지역/모멘츠 관리 개선 통합본
- 정확한 위치/지도/실시간 위치 기능은 넣지 않고, 프로필에 `서울/부산/경기` 같은 넓은 지역만 저장하도록 변경.
- 친구 화면에 `같은 지역 친구 보기` 섹션 추가. 이미 친구인 사람은 후보에서 제외.
- 초기 프로필/프로필 수정에 넓은 지역 선택 추가.
- 모멘츠는 영상 제외, 사진 전용 유지.
- 모멘츠 수정 시 기존 사진을 개별 삭제하고 새 사진을 추가할 수 있도록 개선.
- 서버의 모멘츠 수정 API가 `media` 배열 변경을 저장하도록 개선.
- 모멘츠 사진 최대 5장 저장으로 프론트/서버 제한 통일.
