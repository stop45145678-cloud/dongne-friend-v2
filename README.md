# 동네친구 v2 - Moments Main / 5-Minute Talk / Anonymous Note

데이팅 앱 톤을 제거하고, 일상 공유와 부담 없는 대화를 중심으로 바꾼 친구 찾기 앱 버전입니다.

## 핵심 방향

- 일상 공유, 공감, 대화 매너 중심
- 메인은 모멘츠 피드
- 2순위는 5분 대화
- 3순위는 익명 한마디
- 맵 구조 제거

## 주요 기능

- 모멘츠
  - 하루 일과, 기분, 사진 URL, 음악 링크 공유
  - 전체 공개 / 친구만 공개
  - 공감, 짧은 답장
- 익명 한마디
  - 랜덤으로 부담 없는 글 보내기
  - 답장 주고받기
  - 서로 공감하면 친구 연결 가능
- 5분 대화
  - 자동 종료되는 짧은 대화
  - 2시간 재연결 제한
  - 거절해도 온도 변화 없음
  - 대화 후 매너 평가로만 온도 상승
- 친구 목록
- 매너 온도 랭킹
  - 대화 매너 참고용
- 활동/초대 보상
- 표현 상점

## 실행

```bash
npm install
npm run dev
```

서버는 `server/index.js`, 프론트는 `src` 폴더에 있습니다.


## 안전/심사 대응 기능

- 차단: `/blocks/:userId`로 차단/해제, 친구/추천/랜덤 연결에서 제외
- 신고: 모멘츠/댓글 신고 누적 시 자동 숨김, 5회 누적 시 온도 하락
- 콘텐츠 필터: 욕설, 성희롱, 조건만남, 협박, 개인정보 요구성 표현을 게시글/댓글/채팅/익명 한마디에서 차단
- 운영자 처리: `ADMIN_TOKEN` 설정 후 `/admin/moderation`에서 신고 목록 확인, 모멘츠 숨김, 사용자 이용 제한 가능
- 개인정보/약관/커뮤니티 가이드/계정 삭제 요청: `/privacy`, `/terms`, `/community`, `/delete-account` 제공

출시 전 개인정보처리방침의 사업자명, 연락처, 데이터 보관 기간, 위탁/국외 이전 여부는 실제 운영 정보로 교체해야 합니다.
